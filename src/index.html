MessageSecret

Te damos la bienvenida a la aplicación web que te permitirá cifrar y/o descifrar según sea tú necesidad, utilizamos el metodo César con el cual solo ingresas el mensaje, el número de desplazamiento y voila!! Tendras tú mensaje listo para compartir. De lo contrario si recibiste un mensaje encriptado, descifralo.
